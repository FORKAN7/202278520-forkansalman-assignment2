## Assignment 4 – Personal Web Application

## Project Description

This project is a complete and polished personal web application developed for SWE363 Assignment 4.

It builds on my previous work from Assignments 1, 2, and 3.  
- Assignment 1 focused on the basic structure and layout  
- Assignment 2 introduced interactivity and user-focused features  
- Assignment 3 added advanced logic, API integration, and state management  
- Assignment 4 finalizes the project into a professional, production-ready portfolio  

This final version integrates all concepts learned in the course into a single, responsive, and feature-rich web application. It demonstrates front-end development skills, JavaScript logic, API usage, state management, user experience design, and proper documentation.

---

## Features

### Core Features (From Previous Assignments)
- Responsive layout (desktop, tablet, mobile)
- Modern and clean UI design
- Dark mode toggle with saved preference using localStorage
- Personalized visitor name saved using localStorage
- Dynamic greeting message based on time
- Project filtering by category
- Contact form validation with user feedback
- Smooth animations and transitions

---

### Advanced Features (Assignment 3 & 4)
- GitHub API integration to dynamically display repositories
- Project sorting (newest, oldest, title)
- Saving filter and sort preferences using localStorage
- Expandable project details for better information display
- Enhanced validation logic (including message length and email format)
- Improved error handling and user feedback messages

---

### New Enhancements (Assignment 4)
- Professional UI refinement and layout consistency
- Improved project presentation (images, tags, details, links)
- Skills section highlighting technical abilities
- Cleaner and more organized code structure
- Optimized CSS (removed unused styles)
- Improved navigation and user flow
- Enhanced documentation and explanation of features
- Better GitHub integration (showing selected/highlighted repositories)

---

## Technologies Used

- HTML5 (semantic structure)
- CSS3 (Flexbox, Media Queries, CSS Variables)
- JavaScript (DOM manipulation, events, logic)

## User Guide

1. Enter your name and save it to personalize your experience  
2. Toggle between light and dark mode  
3. Navigate to the Projects section  
4. Use filters to display specific categories  
5. Sort projects by date or title  
6. Click “View Details” to see more information  
7. Explore GitHub repositories loaded dynamically  
8. Use the contact form with proper validation
